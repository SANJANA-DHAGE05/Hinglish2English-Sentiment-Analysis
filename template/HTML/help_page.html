FAQ


File Upload

How to select and submit file from your device?
Click on select file button and select file from your device and click on translate and your file will be translated.
Which file format is supported?
Only text(.txt) format is supported.


Voice Input

How to give voice input?
Click on mike symbol and record your voice ,after voice record you can click on translate button to submit your voice.
Can we able to hear output in voice?
Yes..You can click on speaker symbol which is at the bottom of the output textbox to hear your output in voice.
